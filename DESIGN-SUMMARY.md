# Design summary

- Design concept: Employment planning office
- Layout personality: professional consulting
- Typography: Crimson Text headings with Open Sans body text
- Color palette: primary #1e40af, secondary #60a5fa, accent #10b981, background #eff6ff, text #172554, muted #64748b
- Shape language: Slight rounding
- Visual motif: memo blocks and hiring steps
- Unique layout choices: first sections use Employment memo, then a custom service index, process ribbon, question block, and resource strip with varied widths and alignments.
- Difference from previous sites: this redesign changes the hero composition, section ordering, image treatment, card style, spacing rhythm, button shape, and footer treatment for this domain.
